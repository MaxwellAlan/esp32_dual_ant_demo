# ESP32 Dual Antenna Demo

## Env Setup
- IDF Version: release/v4.4 & master
- Board: ESP32-WROOM-DA

## Usage

### Build & Flash
- idf.py flash monitor -p /dev/ttyUSB?

### Support Commands

- ant -i <internal ant module> -o <outside ant num>
- scan [ssid]
- sta <ssid> [password]
- ap <ssid> [password]
- query
